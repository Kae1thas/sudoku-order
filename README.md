# Sudoku Lite для Яндекс Игр

Готовый минимальный HTML5-проект судоку.

## Что внутри
- `index.html` — главная страница игры.
- `styles.css` — стили.
- `game.js` — игровая логика.

## Как запустить локально
### Вариант 1: просто открыть
Можно просто открыть `index.html` в браузере.

### Вариант 2: через локальный сервер
Лучше так, если хочешь тестировать ближе к публикации.

Пример через Python:
```bash
python -m http.server 8000
```
Потом открыть:
```text
http://localhost:8000
```

## Что загружать в Яндекс Игры
1. Положи все файлы в одну папку.
2. Заархивируй содержимое папки в ZIP.
3. Убедись, что `index.html` лежит в корне архива.
4. Загрузи архив в консоль Яндекс Игр.

## Что можно улучшить потом
- генератор уникальных судоку, а не один базовый solved-grid;
- заметки карандашом;
- сохранение прогресса;
- реклама через Yandex SDK;
- таблица рекордов;
- экран паузы/победы;
- адаптация под вертикальный мобильный режим.

## Важно по SDK
Для публикации на Яндекс Играх SDK должен быть подключен корректно.
Для собственного домена используется абсолютный путь к SDK, а при загрузке архива на сервер Яндекса рекомендован относительный путь `/sdk.js`.
